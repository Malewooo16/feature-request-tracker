# Orders Reports

1. Create Reports Routes

2. Download JSPDF

3. In @Reports Create tabs for managing the following reports Orders, Purchases, Inventory, Accounting

4. Each Tab should have select for report type and date range for the report we need

5. The here are the following order reports we need (focus is on paid orders)

## Order Summary
Fields: Order #, Date, Number of Items(Items), Price, Waiter

## Order Detailed
Fields: Date, Item, Quantity, Price

## Payments
Fields: Order #, Date, Price, Payment Method, Waiter

## Refund
Fields: Order #, Date, Item, Price, Reason, Waiter
